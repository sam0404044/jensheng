# 人生製藥形象網站（HTML + CSS + JS）

依「人生製藥形象網站需求確認」架構：**讓產品完整被看見**、呈現公司歷史與品質精神、架構簡單清楚好閱讀。

## 網站結構

| 頁面 | 說明 |
| --- | --- |
| **index.html** | 一頁式首頁：包含首頁介紹、品牌歷史、產品目錄三個區塊，主選單可平滑捲動跳轉至對應區域。 |
| **product-detail.html** | 產品詳情頁：由產品目錄點入，顯示產品名稱、類別、劑型、功效、規格、圖片、外銷資訊。 |

## 產品資料

- **data/products.json**：產品清單。目前為範例 6 筆，可擴充至 300–400 項。
- 每筆產品欄位：`id`, `name`, `nameEn`, `category`, `subCategory`, `dosageForm`, `efficacy`, `specs`, `image`, `exportInfo`。
- 分類 id：`otc`（成藥）、`food`（機能性食品）、`medical`（醫療器材）、`cosmetic`（化妝品）。
- 公司提供完整產品與歷史資料後，只需更新 **data/products.json**（及必要圖片路徑），不需改程式即可擴充目錄。

## 導覽

全站主選單：**首頁 | 品牌歷史 | 產品目錄**。
